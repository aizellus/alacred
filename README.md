# AlaCred

AlaCred is an Alabama-focused trust, credentialing, and designation organization for small businesses.

This repository now contains two layers:

1. A launch-ready static marketing site in the repo root:
   - `index.html`
   - `styles.css`
   - `main.js`
2. The underlying strategy and research used to shape the offer, pricing, standards, and messaging:
   - `AlaCred-Overall-Strategy.md`
   - `01-Credential-Ideas` through `10-Brand-And-Naming`

## Preview

Run a simple local server from this folder:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.

## Research Flow

Suggested working sequence for the source material:
1. Start with `AlaCred-Overall-Strategy.md`.
2. Then move into `01-Credential-Ideas` and `02-Designation-Ideas`.
3. Use `03-Program-Architecture` and `05-Scoring-Standards` to decide how credentials are earned.
4. Use `06-Membership-Packaging` and `07-Sales-And-Marketing` to define revenue and sales motion.
5. Use `08-Operations` and `09-Templates` to operationalize reviews and renewals.

## Notes

- This workspace is strategic and marketing-focused, not legal advice.
- Pricing, standards, and launch language should be validated before public rollout.
